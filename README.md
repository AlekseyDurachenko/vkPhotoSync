vkPhotoSync
===========

VKontakte Album Downloader

![Screenshot](https://raw.github.com/AlekseyDurachenko/vkPhotoSync/master/screenshot.png)

Compiling
---------

    cd libs
    ./download_and_compile.sh
    cd ..
    qmake
    make -j 3
    # starting:
    ./bin/vkphotosync


3rd libraries:
--------------
* Qt - http://qt-project.org/
* Qt-Json - https://github.com/ereilin/qt-json

Icons:
------
